# Hypotenuse Calculator

Calculates the hypotenuse of a right triangle using the Pythagorean theorem: **c = √(a² + b²)**

## Usage

1. Enter side length A
2. Enter side length B
3. Get hypotenuse length
# example
Enter side A: 3
Enter side B: 4
Hypotenuse: 5.0 cm


## Run
javac Main.java && java Main
